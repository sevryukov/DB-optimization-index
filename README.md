# DB Optimization Index

Интерактивный учебный проект по индексам в базах данных.

## Описание

Проект представляет собой одностраничное приложение (`index.html`), которое визуально объясняет:
- назначение индексов
- типы индексов
- влияние индексов на производительность запросов
- лучшие практики использования индексов

## Как запустить

1. Откройте `index.html` в браузере.
2. При необходимости запустите локальный веб-сервер для поддержки браузерных ограничений (например, `python3 -m http.server`).

## Структура

- `index.html` — основная страница с интерфейсом и стилями.
